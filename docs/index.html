<whisper-transcript>

This is an example of using the whisper-transcript web component to display a Whisper transcript that was generated using word timestamps in order to visualize the confidence scores in the context of the words and their sound:

      
      $ whisper --model tiny audio.mp3 --word_timestamps True

Running this command generates an audio.json file which contains detailed timestamp information for each word, and also a probability score that indicates the confidence in Whisper's transcription.
